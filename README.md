# Sponsor Kids 212 🇲🇦

**Helping Children in Morocco** — A community initiative by Moroccans abroad.

> *"Be the blessing you once needed."*

Sponsor Kids 212 supports vulnerable children in Morocco — orphans, children with disabilities, those with absent or disabled parents, and children in extreme poverty — with clothing, gifts, school supplies, medical help, and more.

## 🚀 How to Deploy on GitHub Pages

### Step 1 — Create a GitHub account
Go to [github.com](https://github.com) and sign up if you don't have an account.

### Step 2 — Create a new repository
- Click the **+** icon → **New repository**
- Name it: `sponsorkids212`
- Set it to **Public**
- Click **Create repository**

### Step 3 — Upload the files
Click **uploading an existing file** and upload all of these files:
```
index.html
SponsorKids212Logo.png
favicon.ico
favicon-32x32.png
apple-touch-icon.png
```

Commit with a message like `Initial site launch` and click **Commit changes**.

### Step 4 — Enable GitHub Pages
- Go to your repository **Settings**
- Click **Pages** in the left sidebar
- Under **Source**, select **Deploy from a branch**
- Set branch to `main` and folder to `/ (root)`
- Click **Save**

### Step 5 — Visit your site
After 1–2 minutes, your site will be live at:
`https://DoneByAdam.github.io/sponsorkids212/`

## ✏️ How to Update the Page

To edit content (text, payment details, photos, receipts):

1. Go to your repository on GitHub
2. Click on `index.html`
3. Click the **pencil icon** (Edit)
4. Make your changes
5. Click **Commit changes**

The site updates automatically within seconds.

### Adding photos to the Gallery
Replace the placeholder `<div class="gallery-placeholder">` blocks in `index.html` with:
```html
<img src="YOUR-PHOTO-FILENAME.jpg" alt="Description" style="width:100%; border-radius:14px; object-fit:cover; aspect-ratio:4/3;">
```
Then upload the photo file to the repository.
